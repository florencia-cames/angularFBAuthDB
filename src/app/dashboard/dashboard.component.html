Firebase brings us a great option for editing elements within an array called arrayUnion, which allows us to add new elements to an array without having to search for or remove existing elements.
However, one limitation of arrayUnion is that it does not accept duplicate values. For example, if you try to add the value 50 to an array twice, only one 50 will be stored.

The good thing about using Firebase's array functions is that we have the option to use arrayRemove and only send the number we want to remove, without having to calculate and remove the numbers ourselves from the array.

Increment value

Another utility that we have is called increment, which works similarly to the previous option. We don't need to calculate the value locally, as increment accepts a number and increments the value by that variable.
